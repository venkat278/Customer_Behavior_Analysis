Data Analytics Project Showcasing Customer Behavior Analysis Using Python, SQL and Power BI
1️⃣ Data Loading (Python)
Imported dataset using Pandas
Checked dataset shape and structure
Inspected data types and summary statistics
2️⃣ Exploratory Data Analysis (EDA)
Identified missing values
Checked data distributions
Performed univariate and bivariate analysis
Visualized key insights using charts
Key EDA steps:
.info()
.describe()
Value counts
Correlation analysis
Visualizations (bar charts, histograms, heatmaps)
3️⃣ Data Cleaning
Handled missing values
Removed duplicates
Corrected inconsistent data types
Standardized column formats
Treated outliers (if applicable)
Result: Clean and analysis-ready dataset

Loaded cleaned data into PostgreSQL server
Wrote SQL queries to extract business insights
Example SQL operations:
SELECT
WHERE
GROUP BY
ORDER BY
JOIN
Aggregate functions (SUM, AVG, COUNT)
This step demonstrates database querying and analytical thinking.

Created data model

Dashboard features:
KPI cards
Filters/Slicers
Trend analysis
Category-wise breakdown
Visual insights for decision-making
